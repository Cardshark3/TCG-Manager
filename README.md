# TCG-Manager

This manager is a simple project I wanted to make to help manage my TCG collection. As of now it only manages Pokemon cards, but i have plans to have it work with other tcgs eventually, however that is not my main goal.
This uses the .net MAUI framework for crossplatform development with the main focus for Windows and Android, with currently no plans for officially supporting other operating systems as I have no way to test if they work.

To get started navigate to the Add Card page in the tool bar and fill in the information including: Name, Pack name, Card ID, if the card is a promo, and finally the amount of the card you have in your collection.

once you have Added your card navigate back to the main page and you will see your card on the grid, from here you can select your card and see the information such as the current average market price.
